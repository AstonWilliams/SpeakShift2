# SpeakShift

## Offline Creator Studio for Video, Transcription, Diarization, and Translation

**One desktop app. End-to-end content workflow. Local-first by design.**

SpeakShift is built for creators and teams who are tired of stitching together multiple tools just to go from raw media to publish-ready output.

If you only need raw text from audio, many apps can do that.
If you need **production workflow** (prep -> transcribe -> speaker separation -> translate -> export), SpeakShift is the better fit.

## Core Product Pitch

### 1) Video and Audio Preparation
- Smart aspect workflows for short-form and standard formats
- Resolution targeting and conversion options
- Brightness, contrast, saturation, and visual filters
- Audio cleanup controls (denoise, dehummer)
- Text and emoji overlay support for quick social edits

### 2) Transcription Engine
- Local media transcription from common audio/video formats
- Whisper model range from lightweight to high-accuracy options
- Language auto-detect and manual language selection
- YouTube ingestion workflow and local processing path
- Live recording pipeline for microphone/system-audio capture use cases

### 3) Speaker-Aware Workflows
- Diarization pipeline for multi-speaker content
- Speaker rename and filtering in detail views
- Strong fit for interviews, podcasts, and panel recordings

### 4) Translation Pipeline
- NLLB-based translation flow integrated into transcription lifecycle
- Broad language catalog for source/target selection
- Edit, save, and export translated content

### 5) Batch Throughput
- Queue-based processing with folder/archive/link sources
- Per-queue or global processing profiles
- Runtime-aware recommendations for stability and speed

### 6) Export and Delivery
- Subtitles: SRT, VTT
- Text/data: TXT, JSON, CSV, TSV, Markdown
- Document output where supported: PDF, DOCX
- Copy and native save flows for fast handoff
